# Personal Finance Tracker

Aplikasi pencatat keuangan pribadi berbasis web untuk mengelola pemasukan dan pengeluaran, mengatur budget bulanan, dan melihat ringkasan keuangan.

## 🚀 Fitur Utama
- Login & Register
- Catat pemasukan & pengeluaran
- Atur budget bulanan
- Ringkasan per kategori
- Grafik pengeluaran
- Transaksi berulang

## 🛠️ Teknologi
- Python 3.x
- Flask
- SQLAlchemy
- SQLite
- Jinja2 Templates
- HTML, CSS, JavaScript
